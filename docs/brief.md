# Project Brief: duantoan

Tài liệu này tóm tắt ngắn gọn ý tưởng, bối cảnh và mục tiêu cốt lõi của dự án **duantoan**. Đây là điểm khởi đầu cho AI Agent hiểu được khách hàng muốn làm gì.
